# Gazarzar Land Marketplace - Ажиллуулах заавар

Энэхүү төслийг локал орчинд ажиллуулахын тулд танд дараах зүйлс хэрэгтэй:

### 1. Шаардлагатай зүйлс
- **Python 3.x:** Сервер ажиллуулахад ашиглана. (Ямар нэгэн нэмэлт сан `pip install` хийх шаардлагагүй).

### 2. Ажиллуулах дараалал
1.  Терминал (Terminal/CMD) нээгээд төслийн хавтас руу орно.
2.  Дараах командыг ажиллуулж mock серверийг асаана:
    ```bash
    python mock_server.py
    ```
3.  Сервер ассаны дараа хөтөч (Browser) дээрээ дараах хаягийг нээнэ:
    [http://localhost:8080](http://localhost:8080)

### 3. Функцууд
- **Нэвтрэх:** Утасны дугаараа оруулаад `123456` кодыг ашиглан нэвтэрнэ.
- **Хайлт:** Хотын нэр (Улаанбаатар, Эрдэнэт, Дархан) болон үнэ, хэмжээгээр шүүх боломжтой.
- **Зар оруулах:** 4 алхамтай зар оруулах хэсэг ажиллана. Анхны зар үнэгүй, дараагийн зар 5,000₮-ийн Boost-тэй байна.

---
**Анхааруулга:** Хэрэв 8080 порт завгүй байвал `mock_server.py` файл доторх `PORT = 8080` гэсэн хэсгийг өөрчлөөд дахин ажиллуулаарай.
